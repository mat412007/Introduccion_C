¿Que es un lenguaje de programación?¿En qué consiste la diferencia entre un lenguaje de alto nivel y uno de bajo nivel?

Un lenguaje de programación es el que usamos para comunicarnos con la computadora o el dispositivo que estemos usando, e indicarle lo que queremos hacerla hacer.
Uno de alto nivel se caracteriza por tener una estructura similar a la del lenguaje de los humanos, así que es más fácil escribirlo.
Uno de bajo nivel es más parecido al lenguaje de la computadora, el binario, es más universal, pero cuesta escribirlo.

Lenguajes compilados: C, C++, Erlang, Haskell, Rust.
Lenguajes interpretados: PHP, Ruby, Python, JavaScript, Lua.


¿Qué hace un compilador?

Un compilador es un programa de computadora que transforma código escrito en lenguaje de alto nivel, a código ejecutable, binario. Toma lenguaje de programación, y lo traduce a 1 y 0.

Las 2 herramientas básicas que necesitamos son: editor de código y compilador.

Llenar los espacios en blanco.

  1. Un lenguaje de programación se traduce a código máquina para generar un archivo ejecutable.
  2. En C, cuándo la función main devuelve cero, estamos indicando que el programa terminó sin errores.
  3. En la función printf el primer argumento se conoce como cadena de formato. Dentro del mismo se usa el caracter % para indicar los lugares donde se reemplazan valores o placeholders.
  4. A menudo, los programas de los lenguajes scripting como Bash y Python se denominan script.
  5. El comando gcc hola.c compila un archivo código fuente llamado hola.c a un ejecutable llamado hola.

Indicar verdadero o falso.

  1. La función printf("mensaje") imprime a salida estándar el mensaje seguido de una nueva línea. F
  2. El compilador de C++ es g++ en Linux. V
  3. Java es un lenguaje interpretado. F
  4. JavaScript y Java son sinónimos. F
  5. El compilador no puede anticipar los errores en tiempo de ejecución. F
  6. Un error de tipeo a veces puede producir errores en tiempo de compilación. V

Sin ejecutarlo en la computadora. ¿Cuál es la salida del siguiente programa?

int main(void) {
printf("A ");
printf("B ");
printf("C ");
printf("D\n");
return 0;
}

Respuesta: A B C D
